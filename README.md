# Multi Containers App

This is a repo for new users getting started with Docker.

You can try it out using the following command.

```docker compose up -d```

And open http://localhost:3000 in your browser.

## Bento UI

A modern Japanese-inspired UI design with glassmorphism effects and AI elements is available at http://localhost:3000/bento

Features:
- Grid-based layout inspired by Japanese bento boxes
- Glassmorphism design with blur effects
- AI-themed components and icons
- Responsive design that works on mobile, tablet, and desktop
- Japanese design elements like kanji characters and wave patterns

## Bento Layout System

An interactive dashboard with a responsive Bento-style grid layout system is available at http://localhost:3000/bento-layout

Features:
- Modular grid system with dynamic widgets
- Drag-and-drop functionality for repositioning widgets
- Resizable widgets with resize handles
- Widget management (add, edit, remove)
- Theme options for widgets
- Responsive design with automatic reflow across devices
- State management for widget positions and dimensions
- Visual feedback during drag and resize operations

## Merged Bento Dashboard

The ultimate Bento experience combining the beautiful visual design of the Bento UI with the interactive functionality of the Bento Layout System is available at http://localhost:3000/bento-merged

Features:
- Japanese-inspired glassmorphism design with beautiful animations and effects
- Full drag-and-drop widget management system
- Interactive widgets with 3D tilt effects and animations
- Widget customization with themes, icons, and kanji characters
- Resizable widgets with visual feedback
- Dark mode toggle
- Fluid animations including floating effects and entry animations
- Particle background effects for enhanced visual appeal
- Add/remove/customize widgets through an elegant modal interface
- Fully responsive design that works across all device sizes
